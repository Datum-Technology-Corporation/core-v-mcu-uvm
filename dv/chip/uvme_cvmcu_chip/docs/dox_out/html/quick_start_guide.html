Pre-Requisites

The Moore.io Client CLI, used throughout this tutorial is available via "pipx":

pipx install mio-client

Introduction

The CORE-V-MCU UVM Environment has a Test Bench used to verify the CORE-V-MCU Sub-System thoroughly. This guide will walk you through running a Test from this Test Bench.

Installation

  1. Install the CORE-V-MCU Sub-System UVM Environment:
    mio install uvme_cvmcu_chip
  2. Install the CORE-V-MCU Sub-System UVM Test Bench:
    mio install uvmt_cvmcu_chip

Compile and Elaborate

  1. Compile the CORE-V-MCU Sub-System UVM Test Bench:
    mio sim uvmt_cvmcu_chip -C
  2. Elaborate the CORE-V-MCU Sub-System UVM Test Bench with wave capture enabled:
    mio sim uvmt_cvmcu_chip -E -w

Simulate

Launch CORE-V-MCU Sub-System UVM Test "rand_stim" with seed "1":

mio sim uvmt_cvmcu_chip -S -t rand_stim -s 1

Results

The Moore.io CLI will printout the commands to view both the simulation logs (dont forget to check out the "trn_log" directory for transaction logs) and waveforms.

Conclusion

Thats all for this short guide. For more information, including integration, have a look at the User Guide.
